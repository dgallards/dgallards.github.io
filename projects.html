Mis proyectos

Estos son mis proyectos personales. Se pueden consultar y modificar porque están alojados en GitHub.

Algoritmo de teledetección y clasificación de imágenes

Esta aplicación desarrollada en Python utiliza un modelo de machine learning para clasificar imágenes por satélite.

Herramienta de envío masivo de mails

Esta aplicación desarrollada en Java, Swing y JavaMail permite enviar mails personalizados a una lista de direcciones de correo.

Artificial intelligence maze

Este proyecto realizado en C++ permite recorrer una serie de laberintos mediante varios algoritmos de inteligencia artificial. Sus resultados se pueden consultar y visualizar en tiempo real.

Herramienta de cifrado simétrico

Esta herramienta realizada en Java y Swing permite cifrar y descifrar documentos de manera simétrica, utilizando una clave y varios algoritmos de cifrado. Además está localizada y traducida a español, francés e inglés.

Herramienta de cifrado asimétrico

Esta otra herramienta realizada en Java y Swing permite cifrar, descifrar, firmar y autenticar documentos de manera asimétrica, utilizando una clave privada y clave pública. Utiliza RSA y varias técnicas de cifrado.
